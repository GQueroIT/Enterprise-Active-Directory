# Step 05 - Network Configuration and Static IP

## Objective
Configure the domain controller with proper network settings and assign a static IP address.

## What I Completed
- Configured dual network adapters (NAT and Host-Only)
- Assigned a static IP address to the server
- Configured subnet mask, gateway, and DNS settings
- Verified network connectivity

## Outcome
The domain controller was configured with a stable network setup required for Active Directory.
